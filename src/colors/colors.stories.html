OpenMRS colors

OpenMRS colors are css variables that can be used by any code at any time.

Available colors

--omrs-primary-blue
--omrs-text-black
--omrs-inactive-grey
--omrs-background-canvas

Code example

Some content in a div
/* in a css file */ .my-css-class { background-color: var(<%- chosenColor %>); color: white; width: 10rem; height: 4rem; }
